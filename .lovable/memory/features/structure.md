## Routes
- `/` — Landing page (hero, how it works, matches, testimonials, FAQ, founder note, footer)
- `/search` — Discovery page with filters (price, age, ratings, type), list/map views
- `/provider/:slug` — Provider detail (Q&A, reviews, rates, location)
- `/book/:slug` — Booking flow (date/time → child → confirm/pay)
- `/quiz` — Parent quiz (5 steps → match results)
- `/parent/dashboard` — Parent dashboard (live updates, bookings, children)
- `/parent/messages` — Messaging (chat list + conversation)
- `/provider/dashboard` — Provider dashboard (attendance, children, activity logging, billing, availability)
- `/admin` — Admin panel (approvals, providers, bookings, commission settings)

## Business Model
Hybrid marketplace + SaaS: commission per booking + provider subscriptions (Basic free, Pro $29, Enterprise $79)
Target: both home daycare providers and daycare centers
